Component: tos.chips.sht11.SensirionSht11ReaderP

generic module SensirionSht11ReaderP()

SensirionSht11ReaderP transforms the HAL-level SensirionSht11 interface into a pair of SID Read interfaces, one for the temperature sensor and one for the humidity sensor. It acquires the underlying resource before executing each read, enabling arbitrated access.
